まず `npm run build` を実行してビルドが通ることを確認する。

ビルドが失敗した場合はエラー内容を報告し、コミットは行わない。

ビルドが成功した場合は、CLAUDE.mdのコミットルールに従ってコミットを実行する：
- 変更を論理的な単位に分けて、複数のコミットに分ける（まとめてコミットしない）
- 関連のない変更は別々のコミットに分ける
- コミットメッセージは日本語で、feat:/fix:/improve:/docs: のプレフィックスを付ける
- 何を変更したかではなく「なぜ変更したか」を重視する
- このコマンドを実行したこと自体がユーザーのコミット許可とみなす（追加確認不要）
